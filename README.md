This is a simple and efficient Task Manager project developed to help users organize and manage their daily tasks.

The application allows users to add, update, and delete tasks easily. It is designed with a user-friendly interface and focuses on improving productivity by keeping track of pending and completed tasks.

Key Features:
- Add new tasks
- Update existing tasks
- Delete tasks
- Mark tasks as completed
- Simple and clean interface

Technologies Used:
- Python

Purpose:
This project was created as a practice project to improve programming skills and understand how real-world applications manage user tasks.

Future Improvements:
- Add Graphical User Interface (GUI)
- Integrate database for better data storage
- Add user authentication system
- Improve overall design and user experience
  

This project is beginner-friendly and useful for understanding basic application logic.
